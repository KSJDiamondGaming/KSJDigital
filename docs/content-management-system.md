# Content Management System (CMS)

## Core Principle

Clients can edit content.

Clients cannot edit layouts, styling, branding, components, or source code.

This keeps every website stable while still allowing clients to manage their own content.

## Locked Content

The following must remain controlled by KSJ Digital:

- Layout Structure
- Components
- CSS Styling
- Theme System
- Branding Rules
- Navigation Logic
- Security Logic
- Backend Logic
- Source Code

---

## Editing Workflow

1. Client edits content.
2. Changes are saved as a draft.
3. Client submits publish request.
4. Website Manager reviews changes.
5. Publish approval occurs.
6. 48-hour restore backup is created.
7. New version becomes live.
